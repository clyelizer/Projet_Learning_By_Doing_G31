# Recherche sur les Bases de Données Agricoles Africaines

## 1. Données de Sol (Pédologie)
Les sources les plus fiables pour le continent africain sont issues du projet AfSIS (Africa Soil Information Service).

*   **Africa Soil Profiles Database (AfSP)** : Contient des milliers d'enregistrements de profils de sol géo-référencés avec des analyses chimiques (pH, NPK, matière organique).
    *   *Accès* : [ISRIC Africa Soil Profiles](https://isric.org/projects/africa-soil-profiles-database-afsp)
*   **SoilGrids (ISRIC)** : Cartes mondiales à 250m de résolution, incluant l'Afrique, pour le pH, l'azote, le carbone organique, etc.
    *   *Accès* : [SoilGrids.org](https://soilgrids.org/)

## 2. Exigences des Cultures (Besoins NPK et pH)
Basé sur les publications de la FAO, de l'ISRIC et des recherches sur l'Afrique de l'Ouest (OFRA - Optimization of Fertilizer Recommendations in Africa).

| Culture | pH Optimal | Azote (N) | Phosphore (P) | Potassium (K) | Source |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Maïs** | 5.5 - 7.5 | Élevé (120-150 kg/ha) | Moyen (40-60 kg/ha) | Moyen (40-60 kg/ha) | [1][2] |
| **Riz (Pluvial)** | 5.0 - 6.5 | Moyen (60-90 kg/ha) | Moyen (30-50 kg/ha) | Moyen (30-50 kg/ha) | [1][2] |
| **Manioc** | 4.5 - 7.0 | Faible/Moyen (40-60 kg/ha) | Moyen (30-40 kg/ha) | Élevé (80-120 kg/ha) | [3] |
| **Sorgho** | 5.5 - 8.5 | Moyen (60-80 kg/ha) | Faible/Moyen (20-40 kg/ha) | Moyen (20-40 kg/ha) | [1] |

*Note : Les valeurs NPK sont des recommandations d'apport pour des rendements cibles en Afrique subsaharienne.*

## 3. Données Climatiques
*   **NASA POWER** : Fournit des données météorologiques quotidiennes (température, précipitations, humidité) via API pour n'importe quel point GPS.
    *   *Lien* : [NASA POWER API](https://power.larc.nasa.gov/docs/services/api/)
*   **CHIRPS** : Données de précipitations à haute résolution pour l'Afrique.
